## 项目介绍 ##
	
	单页应用前端开发框架
	基于 react | redux | webpack | ES6

## 目录结构 ##

	--app/——————————————————————————————————————————————————————————————app主目录
		--Actions/
		--Components/
		--Constants/
		--Pages/
		--Reducers/
		--Routers/
		index.js
	--common/———————————————————————————————————————————————————————————公共目录
		--javascript/
		--style/
		index.html
	--configs/——————————————————————————————————————————————————————————配置目录（主要js配置）
		config.build.js
		config.routers.js
		config.user.js
	--libs/—————————————————————————————————————————————————————————————第三方库
		--javascript/
			index.js
	--task/—————————————————————————————————————————————————————————————任务目录（初始化等工作可以在这里加上）
	.babelrc
	package.json
	README.md
	webpack.config.js

## 使用 ##

	· 进入项目文件夹根目录下
	· npm install
	· npm run dev (npm run prod)
